IoT Server API A RESTful IoT Server

The IoT Server data upload service

Endpoints

  • Search channels
    GET /api/search/channel/{keyword}
  • Movies by director
    GET /api/director/{director_name}
  • Movie by IMDB code
    GET /api/movie/{imdb_number}
  • Top 10 Movies (by IMDB score)
    GET /api/movie/top
  • All genres
    GET /api/movie/genre/all
  • Top movies for a given genres
    GET /api/movie/genre/{genre}

Note: This service has limited data. It is for educational purposes only. If you are looking to build a real app based on IoT APIs, you should check out the responder documentation.
